Pong in Pygame
written by Cristopher Hernandez

First player to score over 10 points is the winner

To run the game: execute pong.py (e.g. run 'python pong.py' on the command line)

Player 1 Controls:

'W' -> move up
'S' -> move down

Player 2 Controls:

'Up Arrow' -> move up
'Down Arrow' -> move down
